**🏦 Bank Customer Churn Analysis**

**📌 Project Overview**
This project analyzes customer churn in a retail banking environment using SQL Server and Power BI. The objective is to identify customer segments with high churn risk, analyze customer behavior, and support data-driven retention strategies through an interactive dashboard.

---

**🎯 Objectives**
- Analyze customer churn patterns
- Build an executive Power BI dashboard
- Identify high-risk customer segments
- Generate actionable business insights

---

**🛠️ Tools Used** 

- SQL Server
- Power BI
- DAX
- Microsoft Excel
